# Instagram Engagement Analysis

## Overview
This project focuses on analyzing Instagram engagement data using Python, Pandas, SQLite, Matplotlib, and Seaborn. The dataset contains information about users, photos, likes, comments, follows, and tags. The analysis helps identify user interaction patterns and engagement trends on Instagram.

---

## Objectives
- Analyze Instagram engagement data
- Create and manage a SQLite database
- Perform SQL-based analysis
- Visualize user interactions and content trends
- Generate insights and recommendations for engagement improvement

---

## Dataset
The dataset contains multiple CSV files:

- users.csv
- photos.csv
- tags.csv
- comments.csv
- likes.csv
- follows.csv

---

## Technologies Used
- Python
- Pandas
- NumPy
- SQLite3
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Project Workflow

### 1. Data Import and Cleaning
- Imported multiple CSV datasets
- Cleaned and formatted column names
- Removed duplicates and handled missing values

### 2. SQLite Database Creation
- Created SQLite database
- Converted DataFrames into SQL tables
- Performed SQL queries for analysis

### 3. Exploratory Data Analysis
- Analyzed users, photos, likes, comments, and follows
- Identified engagement patterns
- Studied interaction distributions

### 4. Data Visualization
- Interaction type distribution
- Photo type analysis
- Engagement trend visualizations

---

## Key Insights
- Likes are the most common interaction type
- Certain photo types receive higher engagement
- User activity is highly engagement-driven
- Instagram engagement data can help improve content strategy

---

## Recommendations
1. Post more engaging content types
2. Improve visual quality of posts
3. Maintain consistent posting schedules
4. Analyze audience interaction trends regularly
5. Focus on content that generates higher engagement

---

## Conclusion
This project demonstrates how Instagram engagement data can be analyzed using Python and SQL techniques. The analysis provides useful insights into user behavior, content performance, and engagement trends that can help optimize social media strategies.
